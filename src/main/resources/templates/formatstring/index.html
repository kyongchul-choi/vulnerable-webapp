포맷 스트링 취약점

설명: 포맷 스트링 취약점은 외부 입력값을 그대로 포맷 문자열에 삽입하여 처리할 때 발생합니다. 공격자는 이를 악용해 메모리 접근, 정보 탈취, 또는 시스템 명령 실행 등을 수행할 수 있습니다.

대책: 포맷 문자열에 외부 입력값을 직접 삽입하지 말고, 안전한 형식화 함수 또는 라이브러리를 사용하세요.

샘플 코드

취약 코드:

// C 언어 예제

char userInput[100];

scanf("%s", userInput); // 사용자 입력 받기

printf(userInput); // 취약: 입력값이 포맷 문자열로 사용됨

안전 코드:

// C 언어 예제

char userInput[100];

scanf("%s", userInput); // 사용자 입력 받기

printf("%s", userInput); // 안전: 포맷 문자열을 명시적으로 지정
