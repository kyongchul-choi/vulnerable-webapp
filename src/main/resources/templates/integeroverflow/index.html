정수형 오버플로우

설명: 정수형 오버플로우는 숫자의 범위를 초과하는 값을 처리하려고 할 때 발생합니다. 이는 예상치 못한 동작을 유발할 수 있으며, 공격자가 이를 악용하여 애플리케이션을 손상시킬 수 있습니다.

대책: 값의 범위를 검증하거나 안전한 데이터 타입(Long, BigInteger 등)을 사용하여 정수형 오버플로우를 방지하세요.

샘플 코드

취약 코드:

public void addValues() {

int maxValue = Integer.MAX_VALUE; // 최대값 2,147,483,647

int result = maxValue + 1; // 오버플로우 발생

System.out.println("결과: " + result); // 예상치 못한 값 출력

}

안전 코드:

public void addValuesSafely() {

long maxValue = Integer.MAX_VALUE;

long result = maxValue + 1; // 안전하게 처리

if (result > Integer.MAX_VALUE) {

throw new ArithmeticException("오버플로우 발생!");

}

System.out.println("결과: " + result);

}
