SQL Injection (SQL 삽입)

설명: SQL 삽입은 응용 프로그램의 입력 필드를 통해 SQL 쿼리를 조작하여
데이터베이스에 비정상적인 액세스를 수행하거나 정보를 탈취하는 취약점입니다.

대책: 입력값을 SQL 쿼리에 직접 삽입하지 말고,
PreparedStatement 또는 ORM을 사용하여 안전하게 처리하세요.
