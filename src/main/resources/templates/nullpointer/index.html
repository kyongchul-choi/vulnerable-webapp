널 포인트 역참조

설명: 널 포인트 역참조는 널(null) 값을 참조하려고 할 때 발생하며, 런타임 에러를 초래할 수 있습니다. 이 취약점은 프로그램이 예외를 처리하지 못하고 중단되는 문제를 유발합니다.

대책: 모든 객체와 변수의 null 여부를 확인하거나, Optional 클래스를 사용하여 널 값을 안전하게 처리하세요.

샘플 코드

취약 코드:

public String getName(Person person) {

// 취약: 널 체크 없이 person 객체의 메서드 호출

return person.getName();

}

설명: 위 코드는 person이 null인 경우, `NullPointerException`이 발생하여 프로그램이 비정상적으로 종료됩니다.

안전 코드:

import java.util.Optional;

public String getNameSafely(Person person) {

// 안전: Optional을 사용하여 null 여부를 처리

return Optional.ofNullable(person)

.map(Person::getName)

.orElse("Unknown");

}

설명: 이 코드는 Optional 클래스를 사용하여 person이 null인지 안전하게 확인하고, 널 값일 경우 기본 값을 반환합니다.
