위험한 형식 파일업로드

설명: 위험한 형식 파일 업로드는 공격자가 악성 파일을 서버에 업로드하여 시스템의 보안을 침해하거나 원치 않는 동작을 실행할 수 있는 취약점입니다.

대책:

  • 파일 형식 검증: 업로드된 파일의 확장자를 확인하고, 허용된 파일 형식만 처리하세요.
  • 파일 이름 검증 및 재지정: 업로드된 파일의 이름을 재지정하거나 사용자 제공 이름을 사용하지 않도록 하세요.
  • 파일 실행 방지: 업로드된 파일이 실행되지 않도록 웹 서버에서 적절한 디렉터리에 저장하세요.
  • 파일 크기 제한: 업로드 가능한 파일 크기를 제한하여 대규모 공격을 방지하세요.
  • 안티 바이러스 스캔: 업로드된 파일을 바이러스 스캔하여 악성 파일 여부를 확인하세요.
